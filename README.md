# Product Case Studies

This repository contains selected case studies of SaaS platforms, product systems, and web applications I’ve contributed to.

Some projects are private and owned by employers or clients, so source code cannot be shared publicly. These write-ups focus on architecture, technical responsibilities, and product outcomes rather than proprietary implementation details.

## Case Studies

### Physician Connection Platform

Production SaaS platform coordinating scheduling workflows between pharmaceutical representatives and physician practices.

**Stack**

Next.js · React · TypeScript · Tailwind · OAuth · Vercel

**Highlights**

- Multi-role SaaS architecture for representatives, physicians, and practice administrators
- Appointment request workflows and approval states
- Calendar-driven scheduling coordination
- Authentication and protected route architecture
- Stabilization work supporting MVP readiness and testing
